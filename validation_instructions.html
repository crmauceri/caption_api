Location verification task

Instructions:

We are collecting a dataset which will help teach robots to find objects in cluttered and confusing locations. 

You will be shown a series of images with a description of a single object in that image. Imagine that someone has given you the description and now you need to fetch the object. Your task is to click on the object.

For example, here is a typical image with an object description. 

Description: The binder is on the top shelf in the corner underneath a box.

We asked the annotators who wrote the descriptions to describe the object's location with enough detail that someone else could find it based on their description. Please flag descriptions that don't describe locations or that don't make sense.
 

Validation:

You do not need to correctly identify every object. Because the descriptions are collected from the web, some might be difficult or confusing. If you can't find an object that fits the description, please use the "There is no object which fits the description" option. There should be no need to guess which object the description refers to.

We will insert some questions where other annotators have had high accuracy (gold standard). These questions should be easy to answer correctly. If you have difficulty with these questions, your responses may be discarded.

Completing the Survey:

The survey will store your progress. So feel free to take a break at any time. Just leave the page open and return later. Please complete the survey within 24 hours.
